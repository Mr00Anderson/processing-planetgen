# Planet generator

Made with [Processing](https://processing.org/)

## How to use

* 'h' - show/hide help
* 'z' - enable/disable random planet sizes
* 'q' - scale up
* 'a' - scale down
* 'e' - raise max size
* 'd' - lower max size
* SPACEBAR - reconstruct planets
